Configurations

Property Value Description
title String Title of the table
yAxis 0 to (dataset.length-1) column index of the dataset for Y-axis
xAxis 0 to (dataset.length-1) column index of the dataset for X-axis
pointLabel 0 to (dataset.length-1) column index of the dataset for text labels
width px width of the chart
height px height of the chart
chartType bar Type of the chart
textAngle (-360,+360) Angle between lables and X-axis
barColor color code color of the bars

Sample code for above charts


/******************************************************* Installation of the library ******************************************************/

             <link rel="stylesheet" href="bootstrap.min.css">
             <link rel="stylesheet" href="igviz.css">
             <script src="d3.min.js"> </script>
             <script src="vega.js"> </script>
             <script src="igviz.js"> </script>


/***********************************************************Invoking the Library **********************************************************/

                        var width = document.getElementById("area").offsetWidth; //canvas width

                        var config = {
                            "title": "Stacked Area Sample",
                            "yAxis": 2,
                            "xAxis": 1,
                            "groupedBy":0,
                            "width": width,
                            "height": height,
                            "chartType": "area"
                       }



                        //draw a stacked area chart on div #area
                    chart=igviz.setUp("#area",config,dataTable);
                    chart.plot(dataTable.data)
                    window.onresize=function(){chart.resize()};
            </script>
